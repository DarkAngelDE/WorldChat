# Changelog

All notable changes to WorldChat will be documented here.

## [1.0.0] — Initial Public Release

### Added
- Global cross-zone chat via `.w <message>`
- Per-player opt-in/out with `.wc on` / `.wc off` / `.wc status`
- Class-colored player names
- Faction icons (Alliance / Horde)
- GM/Moderator badge (Blizzard icon)
- Forbidden link detection with automatic muting
- Minimum played time gate (configurable)
- Anti-spam cooldown (configurable)
- SQL table for storing player preferences
